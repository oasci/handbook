# Plane

We use [plane][plane] to manage all of our projects which integrates into GitHub.
You can find the OASCI plane workspace here: [app.plane.so/oasci](plane-oasci).

## Access

Our [plane workspace][plane-oasci] is not available to the public.
However, we have integrated [plane][plane] into our [GitHub][github-oasci] organization.
If a [plane][plane] project is linked to a repository, you can still see our tasks in a transparent manner.
